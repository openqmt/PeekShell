# 赞助

PeekShell 是开源、免费的跨平台 SSH 客户端。你的支持能帮助项目持续维护、修复问题并推进新功能。

## 免费支持

这些方式同样很有价值，且不需要花钱：

- 在 [GitHub](https://github.com/openqmt/PeekShell) 给项目点 **Star**
- 提交 [Issue](https://github.com/openqmt/PeekShell/issues) 反馈问题或建议
- 发起 [Pull Request](https://github.com/openqmt/PeekShell/pulls) 贡献代码或文档
- 向同事或社区推荐 PeekShell
