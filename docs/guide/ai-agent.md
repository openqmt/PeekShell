# AI 助手

PeekShell 的 AI Agent 设计原则是：**模型可以提议，但不能擅自执行**。

## 工作闭环（目标形态）

1. 收集上下文：主机信息、当前工作目录、最近终端输出、你的问题
2. 模型返回结构化结果：说明文字 + 待执行命令列表（含风险等级）
3. UI 展示命令与风险，由你同意 / 拒绝 / 编辑后再执行
4. 在当前 SSH 会话执行，捕获输出
5. 将结果回灌模型，决定结束或继续提议（每一轮仍需确认）

## 安全规则

- 默认禁止自动执行
- 高危命令（如递归删除、改防火墙、写系统文件）应二次确认
- API Key 仅存本地；支持 OpenAI 兼容接口（含本地 Ollama 等）
- 提议与执行应写入本地审计日志（随版本完善）

## 当前状态

设置中可配置 AI 提供商（Base URL / Model / API Key）。完整 Agent 闭环仍在 Phase 2 推进中；请以应用内提示为准。

配置就绪后，可在右侧 AI 面板用自然语言提问，例如「检查 nginx 错误日志」。
