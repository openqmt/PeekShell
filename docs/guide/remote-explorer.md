# 远程文件

连接主机后，可在远程文件浏览器中浏览远端目录树，预览与编辑文件，并完成上传与下载。

## 基本操作

- 进入目录、返回上级、按路径跳转
- 预览文本 / 图片；在线编辑文本并保存回远端
- 上传本地文件、下载远端文件（支持拖放上传）
- 右键菜单：新建、重命名、删除、权限、复制路径等
- 刷新当前目录列表

## 传输任务

工具栏中的传输按钮可打开上传 / 下载面板：

- 查看进行中、排队与已完成任务，以及批量进度
- **停止** / **继续** 批量传输
- **清除已完成** 任务
- 设置默认下载目录（未设置时每次询问保存位置）

## 资源管理器设置

在资源管理器空白处或右键菜单中打开 **更多设置**，可配置：

### 预览文件大小

超过上限的文件无法在应用内预览或在线编辑，需下载到本地后再查看。

### 目录树类型显示

左侧目录树中文件类型的展示方式：

| 选项 | 说明 |
|------|------|
| 文字（DIR / FILE） | 以文字标签区分目录与文件 |
| 图标 | 线框图标，按文件类型着色 |
| Windows 风格 | 彩色图标，接近 Windows 资源管理器观感 |
| macOS 风格 | 接近 Finder 的文件夹与文档图标 |

设置会保存在本地，可随时恢复默认。

## 与终端配合

远程文件适合传输与查看；需要执行命令、查看实时输出时，切换回终端标签即可。两者共享同一 SSH 连接上下文。
