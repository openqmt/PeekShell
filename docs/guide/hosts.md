# 主机管理

在侧栏打开主机列表，可按分组管理 SSH 连接。

## 新增连接

填写：

- **名称** — 列表中显示的别名
- **主机** — IP 或域名
- **端口** — 默认 `22`
- **用户名**
- **所属分组**（可选）
- **备注**（可选）

认证方式二选一：

| 方式 | 需要填写 |
| --- | --- |
| 密码 | 登录密码 |
| 公钥 | 私钥文件路径，可选私钥口令（passphrase） |

凭证优先存入系统钥匙串，避免明文落盘。

## 分组

支持新建、重命名、删除分组。删除分组时，可按提示处理组内主机。

## 连接

在主机列表中选择主机并连接。连接成功后，中间工作区打开终端标签，左侧可查看该主机的系统与资源信息。
