# Terminal

PeekShell renders SSH PTY sessions with xterm.js.

## Multi-tab

Each connected host gets a tab. Switch between them freely; closing a tab disconnects that session.

## Host overview

The left panel (collapsible) shows after connect:

- System: OS, kernel, architecture, uptime
- Resources: memory, swap, disk
- Process summary
- Network interface traffic

Metrics are polled over SSH and can later feed AI context.

## Display settings

Configure which sidebar sections are visible, plus theme (dark / light) and UI language (Chinese / English).
