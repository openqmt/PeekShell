# Remote files

After connecting, use the remote file explorer to browse the remote tree and transfer files.

## Basics

- Enter directories and go up
- Upload local files to a remote path
- Download remote files to your machine
- Refresh the current listing

Available actions follow the current UI; richer permission controls and batch jobs will land along the roadmap.

## With the terminal

Use remote files for transfer and browsing; switch back to a terminal tab when you need to run commands or watch live output. Both share the same SSH connection context.
