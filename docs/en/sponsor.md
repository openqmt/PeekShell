# Sponsor

PeekShell is a free, open-source cross-platform SSH client. Your support helps keep maintenance going, fix bugs, and ship new features.

## Free ways to help

These are just as valuable — no payment needed:

- Give the project a **Star** on [GitHub](https://github.com/openqmt/PeekShell)
- Open an [Issue](https://github.com/openqmt/PeekShell/issues) with bugs or ideas
- Send a [Pull Request](https://github.com/openqmt/PeekShell/pulls) for code or docs
- Recommend PeekShell to teammates and communities
